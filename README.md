# Introducing Blockchain with Java: 
Program, Implement, and Extend Blockchains with Java by Spiro Buzharovski (Apress, February 2022, 188 pages) is a practical, hands-on guide for Java developers to build a fully functional blockchain-based cryptocurrency application from scratch.
The book starts with foundational blockchain concepts—such as encryption, hashing, smart contracts, and Bitcoin basics—then walks readers through implementing core components in Java, including blocks, transactions, and wallets.
It covers:

Setting up a SQLite database with JDBC for persistent storage
Building a user interface using JavaFX
Creating a peer-to-peer (P2P) multi-threaded network
Implementing the service layer for full functionality

Readers end up with a complete cryptocurrency app running on a P2P network, using Java, JavaFX, and SQL.
The book features step-by-step instructions, numerous exercises, test questions, and suggestions for extending the code. It includes access to a GitHub repository with fully working code, making it ideal for building a portfolio project.
Target audience: Developers with basic Java knowledge (or similar OOP language), some familiarity with markup like FXML/HTML, and SQL.

Introduction to Blockchain

Extras

This book emphasizes learning by doing, providing both theory and a complete implementable project without relying on external blockchain platforms.
